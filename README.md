# weather-app

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Type-Check, Compile and Minify for Production

```sh
npm run build
```

### Install http-server globally

```sh
npm install --global http-server
```

### Deploy using http-server

```sh
http-server [path]
```
ex.
```sh
http-server dist
```
